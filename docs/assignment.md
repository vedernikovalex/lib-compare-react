# **Porovnání vybraných React UI knihoven**

###### Comparison of Selected React UI libraries

# Hlavní cíl

Bude provedeno komplexní vybraných React UI knihoven (např. Material UI, Ant Design, Chakra UI) pro vývoj webových aplikací v dané oblasti.

### Dílčí cíl 1

Bude zpracována kritická literární rešerše dostupných studií týkajících se React UI knihoven se zaměřením na jejich srovnání a výkonnostní metriky.

### Dílčí cíl 2

Bude navržena a vyvinuta referenční aplikace, která bude postupně implementována s využitím vybraných knihoven při zachování maximálně identické funkčnosti a designu.

### Dílčí cíl 3

Budou změřeny a vyhodnoceny vybrané metriky např. výkonu, přístupnosti a optimalizace pro vyhledávače, zahrnující zejména Core Web Vitals, velikost přenášených dat a auditní skóre nástroje Lighthouse pro každou implementaci.

### Dílčí cíl 4

Bude provedeno vhodné srovnání , v němž budou stanovena vhodná kritéria na základě odborné literatury a případně konzultace se stakeholdry.

# Metodika

### Teoretická část

Bude provedeno systematické vyhledávání zdrojů ve veřejných databázích a odborných knihách. Budou analyzovány dosavadní přístupy k hodnocení React UI knihoven a formulovány relevantní metriky.

### Praktická část

Bude vyvinuta referenční aplikace v React a poté re-implementována s využitím vybraných knihoven.  
Výkonnost, použitelnost a přístupnost budou měřeny a hodnoceny podle vybraných kritérií.

Na základě syntézy teoretických poznatků a výsledků praktické časti budou formulovány závěry práce.
